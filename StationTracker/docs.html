Station Tracker

Overview

The station tracker addon pulls data from Station Command Center and save the data state inside the addon. The addon generates a client-side webpage that displays the data in a relevent format for the user.

Installation

Installation is very simple. Click the install button below and click Add.

Install Station Tracker

Usage

After the addon is installed, a new addon will be added to the addon bar in the top right hand corner.

Clicking on the puzzle piece icon will load the client-side webpage and load a new minimized window to extract data from Station Command Center. The page will be blank while data is loading from Station Command. After Station Cammand's data has been extracted, the addon will close the new window and update the new client-side dashboard.

Clicking on the the puzzle piece again will repeat the process above with the exception that there wont be a new client-side dashboard. The addon will focus the current dashboard and update that one instead. This is the only way to update data in the dashboard until an auto-update feature is implemented.

Priority List

Once data is showing up in the dashboard, a priority list is generated.

Route View

Station View

The Route View shows all in-progress routes seperated by stage-by times. The wave stage-by times are 5 minutes earlier than Station Command's stage-by times. The routes are prioritized based on completion and can be selected for further details. The stations in Station View are listed in order. Stations will have a red icon if the station isn't at full capacity. Stations can be selected and the first route in the station's list will be displayed.

Filters

You can switch between Route View and Station View in the filter section. The routes can be sorted by percentage completed or by total remaining packages. This is useful when you have smaller routes, such as flex routes, mixed in with large DSP routes. The stations can be filtered with the range slider, too. The slider will only show the stations between the min and max ranges, inclusively.

Station Details

When a route or station is selected in the priority list, its details are displayed in the details tab.

The station details tab shows a breakdown of all the routes that were built during throughout the day. The In Progress section shows each route currently being showed at each station as well as their cart data. The Completed Route section shows completed route, weather they were fully picked or cut, and the status of the carts.
