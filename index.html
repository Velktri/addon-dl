Overview

This is an overview for addons and tools developed for Amazon AR delivery stations. Currently, there are two tools being developed.

Wave Tracker

The Wave Tracker tool is a firefox addon developed for outbound OTR and UTR teams. Documentation and installation can be found in the Wave Tracker tab.

Function

The tool's primary purpose is to keep track of carts and routes in all assigned waves.
